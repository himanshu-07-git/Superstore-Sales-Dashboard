
# 📊 Superstore Sales Dashboard

## 🔍 Project Overview

The **Superstore Sales Dashboard** is a data analytics project designed to analyze sales performance, customer behavior, product trends, and regional insights using real-world retail data.

This project demonstrates end-to-end data analysis skills including:

* Data Cleaning
* SQL Analysis
* Data Visualization
* Business Insights Generation

The dashboard helps businesses monitor performance and make data-driven decisions.

---

# 🛠️ Tools & Technologies Used

* **MySQL** – Data storage and SQL analysis
* **Python** – Data cleaning and preprocessing
* **Pandas & NumPy** – Data manipulation
* **Power BI** – Interactive dashboard creation
* **Excel / CSV Dataset** – Raw data source

---

# 📁 Dataset Information

The dataset contains retail store sales information including:

* Orders
* Customers
* Products
* Sales
* Profit
* Regions
* Categories
* Shipping details

---

# 📌 Project Objectives

* Analyze overall sales and profit performance
* Identify top-performing products and categories
* Track region-wise sales trends
* Understand customer purchasing behavior
* Find loss-making products and areas
* Create an interactive business dashboard

---

# 📈 Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Profit Margin
* Top Selling Products
* Region-wise Revenue
* Monthly Sales Trend
* Category & Sub-category Performance
